# 决策算法 Skill - 产品需求文档

## 产品概述

### 背景

人生重大决策（职业选择、投资决策、创业判断、感情关系等）往往缺乏系统性的分析框架。人们容易陷入情绪化决策、认知偏差或信息过载，导致决策质量不稳定。

### 目标

基于老喻 20 年决策研究，将 EKB 决策算法框架（期望值 + 凯利公式 + 贝叶斯定理）封装为 AI Skill，帮助用户：

1. 建立可重复调用的决策分析流程
2. 识别常见决策陷阱和认知偏差
3. 量化评估决策的期望值和风险
4. 建立可纠错的决策闭环

### 核心价值主张

> 不是替用户做决定，而是帮用户建立可纠错的决策闭环，成为命运的主人。

## 用户画像

### 主要用户

| 特征 | 描述 |
|------|------|
| 年龄 | 25-45 岁 |
| 职业 | 职场人士、创业者、投资者 |
| 特征 | 面临重大决策，希望理性分析而非凭感觉 |
| 痛点 | 决策时容易纠结、内耗、缺乏系统性框架 |

### 使用场景

1. **职业决策**：要不要辞职？要不要跳槽？要不要创业？
2. **投资决策**：这笔投资该不该做？该投多少？
3. **感情决策**：要不要分手？要不要结婚？
4. **消费决策**：该不该买？值不值？
5. **风险决策**：这个风险大不大？该不该承担？

## 功能需求

### 核心功能

#### 1. 决策分析工作流

- **输入**：用户的决策问题（自然语言）
- **处理**：
  1. 识别决策类型
  2. 心法筛查（陷阱识别）
  3. 胜率 + 赔率分析
  4. 期望值计算
  5. 凯利公式（资源配置）
  6. 贝叶斯更新策略
- **输出**：结构化决策建议

#### 2. 决策陷阱识别

自动识别用户是否陷入以下陷阱：
- 内耗陷阱（反复横跳）
- 惯性陷阱（用过去惯性做决策）
- 委托陷阱（误以为委托=放弃决策权）
- 确定性陷阱（追求确定性）
- 成功陷阱（过去成功成为枷锁）
- AI 陷阱（让外部系统替代决策权）

#### 3. 多深度分析模式

| 模式 | 触发方式 | 分析深度 |
|------|---------|---------|
| 快速判断 | "帮我快速评估" | 仅期望值正负判断 |
| 深度分析 | "帮我详细分析" | 完整七步流程 |
| 特定框架 | "用凯利公式帮我算" | 聚焦单一工具 |

#### 4. 决策算法库

内置 20+ 个决策算法，按需调用：
- 先小人后君子、二阶理性、复利再认识
- 处置效应、屁股决定脑袋、满意主义
- 遍历性原则、胶带纸思维、奥卡姆剃刀
- ...（完整列表见 SKILL.md）

### 辅助功能

#### 1. 决策计算器

命令行工具，用于精确计算：
- 期望值（EV）
- 凯利公式（最优下注比例）
- 完整决策分析报告

#### 2. 课程知识提取

从课程材料自动提取算法卡片：
- 支持 TXT/MD 格式输入
- 自动分块处理长文本
- 输出结构化 JSON/Markdown

#### 3. Skill 验证器

验证 SKILL.md 质量：
- 检查 frontmatter 完整性
- 验证核心公式正确性
- 检查算法覆盖率
- 检测内容矛盾

## 非功能需求

### 安全性

- 不鼓励用户投入输不起的资源
- 不鼓励用户对不懂的事情下重注
- 不替用户做人生重大决定
- 涉及违法犯罪明确拒绝
- 纯粹赌博/彩票告知期望值为负

### 可靠性

- 所有公式和规则与课程原文一致
- 验证器确保输出质量稳定
- 版本管理支持回滚

### 可扩展性

- 知识库支持增量更新
- 算法库支持按需扩展
- Prompt 模板支持自定义
