## 1. 数学逻辑审查

### 1.1 期望值 (Expected Value) 公式

**文档表述**（Line 195-196）：
```
期望值 = 胜率 × 收益 - (1-胜率) × 损失
```

**评审结果**：✅ 正确

**分析**：
- 标准期望值公式为 EV = p × G + (1-p) × L
- 当"损失"理解为正值（投入金额）时，文档公式 EV = p × 收益 - (1-p) × 损失 = pG - (1-p)L 等价于 EV = pG + (1-p)(-L)
- 塔勒布案例验证（Line 201）：70%涨1% vs 30%跌10%
  - EV = 0.7 × 1% - 0.3 × 10% = 0.7% - 3% = -2.3% ✅ 正确

**建议**：可补充说明"损失"为正值输入，避免歧义。

### 1.2 凯利公式 (Kelly Criterion) 公式

**文档表述**（Line 210-212）：
```
f* = (p × b - q) / b
其中：p = 胜率，q = 1-p，b = 赔率（赢/亏比）
```

**评审结果**：✅ 正确，但符号说明不够清晰

**分析**：
- 标准凯利公式：f* = (p × b - q) / b
- 其中 b = 赢/亏比（odds），即赔率
- 当 b = 1 时，f* = p - q = 2p - 1（经典形式）
- 六大凯利智慧（Line 221）描述准确

**潜在问题**：
- 文档未明确区分"赔率"（odds）和"收益率"（return）的差异
- 现实中赔率常以"1赔b"形式表达，此时 b > 1
